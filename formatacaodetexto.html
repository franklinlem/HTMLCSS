Principais formatações

Negrito, Destaque, Ênfase e Itálico

Para colocar um termo em negrito usamos a tag B (não semântica). Este é um exemplo de uso do negrito.

Para usar um termo em destaque utilizamos a tag STRONG (semântica). Este é um exemplo de um termo em destaque.

Um termo ficará em itálico usando a tag I (não semântica). Este é um exemplo de itálico.

Para colocar ênfase em um termo usa-se a tag EM (semântica). Este termo é um exemplo de uso da ênfase.

Envelopamento com abreviatura

Selecione um termo que será envelopado. Para utilizar o envelopamento com abreviatura usa-se o atalho CTRL+SHIFT+P (escolha "Wrap with abbreviation/Envelope com abreviatura"). Este termo foi envelopado com a tag I.

Texto marcado

Um texto pode ser marcado pela tag MARK. Este texto foi marcado, por exemplo. Para usar outra cor de marcado usa-se a tag MARK seguida do estilo e cor de fundo desejada. Este texto está marcado com uma cor diferente.

Texto grande e texto pequeno

Para colocar um texto grande usava-se a tag BIG. A tag BIG fica na cor vermelha pra indicar que está obsoleta.

E para colocar um texto pequeno usa-se a tag SMALL.

Texto deletado e texto excluído

Para excluir ou riscar um texto, usa-se a tag DEL. Este texto foi excluído, por exemplo.

Texto inserido e sublinhado

Para inserir um texto é usada a tag INS. Exemplo: Este texto foi inserido aqui.

A tag U é usada para sublinhar um texto quando não há semântica. Aqui está um exemplo de texto sublinhado.

Texto Sobrescrito e Texto Subscrito

Para sobrescrever um termo usamos a tag SUP. Exemplo usado em expoentes matemáticos: 234, 410, 209...

Para subscrever um termo usamos a tag SUB. Exemplo usado em fórmulas químicas: CO2, H2O, H2SO4, HCO3...
